Overview

India’s merchandise trade has grown from under USD 200 billion (2005) to over USD 1.1 trillion (2023), making it the world’s 10th largest merchandise trading nation. This analysis examines total export and import trends, the trade balance trajectory, and the composition of key sectors — cereals, fertilizers, petroleum, and edible oils — using Harmonised System (HS) code classification.

TipKey Findings at a Glance
  • 📈 India’s exports crossed USD 450 billion in 2022–23, up from ~USD 170 billion a decade earlier
  • 📉 Trade deficit is structural and widening: typically USD 200–250 billion annually in recent years
  • 🌾 Rice dominates cereal exports: India is the world’s largest rice exporter, accounting for ~40% of global trade
  • ⛽ Petroleum products are India’s top export category — a testament to refinery capacity expansions
  • 🌱 Fertilizer imports are large and import-intensive: India imports ~$10 billion worth of fertilizers annually
  • 🔄 Export coverage ratio has declined from ~70% (early 2000s) to ~55% (recent years), indicating growing import dependence

2 Data Loading & Cleaning

Code 
usd <- as.data.table(readRDS("/home/pawan/trade/totaltrade/total_exportsusd.rds"))
qty <- as.data.table(readRDS("/home/pawan/trade/totaltrade/total_exportsqty.rds"))
musd <- as.data.table(readRDS("/home/pawan/trade/totaltrade/total_importsusd.rds"))
mqty <- as.data.table(readRDS("/home/pawan/trade/totaltrade/total_importsqty.rds"))
twodigit <- as.data.table(read_excel("/home/pawan/trade/totaltrade/two-digit.xlsx"))

## Standardised cleaning function
clean_trade <- function(dt, val_col) {
  dt[, `:=`(HSCode2 = substr(HSCode, 1, 2),
            HSCode4 = substr(HSCode, 1, 4),
            HSCode6 = substr(HSCode, 1, 6))]
  dt[, (val_col) := as.numeric(gsub(",", "", get(val_col)))]
  dt[is.na(get(val_col)), (val_col) := 0]
  dt
}

usd  <- clean_trade(usd,  "Export")
qty  <- clean_trade(qty,  "Export")
musd <- clean_trade(musd, "Import")
mqty <- clean_trade(mqty, "Import")

## Merge HS classification
twodigit[, HSCode2 := as.numeric(HSCode2)]
qty[,  HSCode2 := as.numeric(HSCode2)]
mqty[, HSCode2 := as.numeric(HSCode2)]
qty  <- merge(qty,  twodigit, by = "HSCode2", all.x = TRUE)
mqty <- merge(mqty, twodigit, by = "HSCode2", all.x = TRUE)

## Aggregate totals
export_usd <- usd[,  .(Export_USD = sum(Export, na.rm = TRUE)), by = Year]
import_usd <- musd[, .(Import_USD = sum(Import, na.rm = TRUE)), by = Year]
export_qty <- qty[,  .(Export_Qty = sum(Export, na.rm = TRUE)), by = Year]
import_qty <- mqty[, .(Import_Qty = sum(Import, na.rm = TRUE)), by = Year]

trade <- Reduce(function(a, b) merge(a, b, by = "Year"),
                list(export_usd, import_usd, export_qty, import_qty))
trade[, `:=`(
  Trade_Balance   = Export_USD - Import_USD,
  Trade_Volume    = Export_USD + Import_USD,
  Export_Coverage = round(Export_USD * 100 / Import_USD, 1),
  Export_Growth   = round((Export_USD / shift(Export_USD) - 1) * 100, 1),
  Import_Growth   = round((Import_USD / shift(Import_USD) - 1) * 100, 1)
)]

       x = "Year", y = "USD Billions",

4 Cereal Exports: Composition

India is the world’s largest rice exporter and a significant wheat exporter in surplus years. The 2022–23 wheat export ban reflects India’s policy of managing domestic food price inflation.

Code 
cereals <- qty[HSCode2 == 10]
cereals[, Commodity := fcase(
  HSCode4 == "1006", "Rice",
  HSCode4 == "1001", "Wheat",
  HSCode4 == "1005", "Maize",
  HSCode4 == "1007", "Grain Sorghum",
  HSCode4 == "1008", "Millet & Buckwheat",
  default = "Other Cereals"
)]
cereals_yr <- cereals[, .(Qty = sum(Export, na.rm = TRUE)), by = .(Year, Commodity)]
cereals_yr[, Total := sum(Qty), by = Year]
cereals_yr[, Share := round(Qty * 100 / Total)]
cereals_yr[, Commodity := factor(Commodity,
  levels = c("Other Cereals", "Millet & Buckwheat", "Grain Sorghum",
             "Maize", "Wheat", "Rice"))]
cereals_yr[, tip := paste0("<b>", Commodity, "</b><br>",
                           "Year: ", Year, "<br>",
                           "Qty: ", format(round(Qty), big.mark = ","), " '000T<br>",
                           "Share: ", Share, "%")]

pce <- ggplot(cereals_yr, aes(x = Year, y = Qty, fill = Commodity,
                               tooltip = tip, data_id = paste(Commodity, Year))) +
  geom_bar_interactive(stat = "identity", position = "stack", width = 0.75, alpha = 0.9) +
  geom_text(aes(label = ifelse(Share >= 8, paste0(Share, "%"), "")),
            position = position_stack(vjust = 0.5), size = 2.8,

  scale_fill_viridis(discrete = TRUE, option = "D") +

  labs(title   = "Cereal Exports from India by Commodity",
       subtitle = "Rice dominates; wheat exports vary by policy and domestic supply conditions",
       x = "Year", y = "Quantity ('000 Tonnes)",
       caption = "Source: Ministry of Commerce | Analysis: Pawan") +
  my_theme

8 Key Takeaways

ImportantSummary of Findings

  1. India’s trade deficit is deep and structural: Petroleum, fertilizers, and edible oils together account for the bulk of the import bill, and all three are heavily import-dependent by structural necessity.

  2. Cereal exports are policy-sensitive: The government alternates between encouraging exports (when there is surplus) and banning them (when domestic prices rise) — illustrating the tension between farmer welfare and consumer welfare.

  3. India is the world’s largest rice exporter: Rice contributes the bulk of cereal export earnings, though the recent quality and residue concerns in some markets require attention.

  4. Petroleum imports are the largest single category: India’s energy import dependence (~85% crude oil from overseas) is both an economic vulnerability and a geopolitical consideration.

  5. The export coverage ratio declining toward 55% suggests imports are growing faster than exports — a medium-term current account concern.

  6. Fertilizer import dependence reinforces the need to expand domestic production capacity and reduce over-subsidisation, which incentivises excessive fertilizer use and soil degradation.

## Key Takeaways

::: {.callout-important title="Summary of Findings"}
1. **India's trade deficit is deep and structural**: Petroleum, fertilizers, and edible oils together account for the bulk of the import bill, and all three are heavily import-dependent by structural necessity.

2. **Cereal exports are policy-sensitive**: The government alternates between encouraging exports (when there is surplus) and banning them (when domestic prices rise) — illustrating the tension between farmer welfare and consumer welfare.

3. **India is the world's largest rice exporter**: Rice contributes the bulk of cereal export earnings, though the recent quality and residue concerns in some markets require attention.

4. **Petroleum imports are the largest single category**: India's energy import dependence (~85% crude oil from overseas) is both an economic vulnerability and a geopolitical consideration.

5. **The export coverage ratio declining** toward 55% suggests imports are growing faster than exports — a medium-term current account concern.

6. **Fertilizer import dependence** reinforces the need to expand domestic production capacity and reduce over-subsidisation, which incentivises excessive fertilizer use and soil degradation.
:::
