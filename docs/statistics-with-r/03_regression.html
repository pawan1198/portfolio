Simple and Multiple Linear Regression in R

Introduction

Regression analysis is a statistical method used to understand and quantify the relationship between variables. It helps us:

  • Explain how a dependent variable changes with one or more independent variables.
  • Predict future values of the dependent variable.

Example: How does a car’s fuel efficiency (MPG) change with its weight and horsepower?

Types of Regression

Type Response Variable Predictors
Simple Linear Continuous 1 continuous
Multiple Linear Continuous 2+ continuous/categorical
Logistic Binary (0/1) Any
Polynomial Continuous Powered terms

This paper focuses on Simple and Multiple Linear Regression.

Simple Linear Regression

Definition

Models the linear relationship between one predictor \(X\) and a response \(Y\):

\[Y = \beta_0 + \beta_1 X + \varepsilon\]

where:

  • \(Y\) = dependent variable (response)
  • \(X\) = independent variable (predictor)
  • \(\beta_0\) = intercept (value of \(Y\) when \(X = 0\))
  • \(\beta_1\) = slope (change in \(Y\) per unit change in \(X\))
  • \(\varepsilon \sim N(0, \sigma^2)\) = random error term

Ordinary Least Squares (OLS) Estimation

OLS minimises the sum of squared residuals:

\[\text{SSR} = \sum_{i=1}^{n}(y_i - \hat{y}_i)^2 = \sum_{i=1}^{n}(y_i - \hat{\beta}_0 - \hat{\beta}_1 x_i)^2\]

The closed-form solutions are:

\[\hat{\beta}_1 = \frac{\sum_{i=1}^{n}(x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^{n}(x_i - \bar{x})^2} = \frac{S_{xy}}{S_{xx}}\]

\[\hat{\beta}_0 = \bar{y} - \hat{\beta}_1 \bar{x}\]

Goodness of Fit — R-squared

\[R^2 = 1 - \frac{\text{SSRes}}{\text{SST}} = 1 - \frac{\sum(y_i - \hat{y}_i)^2}{\sum(y_i - \bar{y})^2}\]

  • \(R^2 = 1\): Perfect fit
  • \(R^2 = 0\): Model explains nothing
  • \(0 < R^2 < 1\): Proportion of variance explained

Multiple Linear Regression

Definition

\[Y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \cdots + \beta_k X_k + \varepsilon\]

In matrix notation:

\[\mathbf{Y} = \mathbf{X}\boldsymbol{\beta} + \boldsymbol{\varepsilon}\]

OLS solution:

\[\hat{\boldsymbol{\beta}} = (\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T\mathbf{Y}\]

Adjusted R-squared

For multiple regression, use Adjusted \(R^2\) to penalise for extra predictors:

\[\bar{R}^2 = 1 - \frac{(1-R^2)(n-1)}{n-k-1}\]

where \(k\) = number of predictors and \(n\) = sample size.

Assumptions of Linear Regression (LINE)

Letter Assumption
L Linearity: relationship between \(X\) and \(Y\) is linear
I Independence: observations are independent
N Normality: residuals are normally distributed
E Equal variance: residuals have constant variance (homoscedasticity)

When and Where to Use

Scenario Use Case
Economics Predicting GDP from investment and consumption
Medicine Predicting blood pressure from age and BMI
Real estate Predicting house price from area and location
Agriculture Predicting crop yield from rainfall and temperature
HR analytics Predicting salary from experience and education

Residual Diagnostics

par(mfrow = c(2, 2))
plot(model_simple, col = "steelblue", pch = 16)

The four diagnostic plots check:

  1. Residuals vs Fitted: Should show no pattern (linearity, homoscedasticity)
  2. Normal Q-Q: Points should follow the diagonal line (normality)
  3. Scale-Location: Should be horizontal (equal variance)
  4. Residuals vs Leverage: Identifies influential points

Coefficient of Determination Summary

models <- list(
  "Simple (wt only)"    = model_simple,
  "Multiple (wt + hp)"  = model_multi,
  "Stepwise selected"   = model_step
)

results <- sapply(models, function(m) {
  c(R2     = round(summary(m)$r.squared, 4),
    AdjR2  = round(summary(m)$adj.r.squared, 4),
    AIC    = round(AIC(m), 2))
})
